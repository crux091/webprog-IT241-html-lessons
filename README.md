# 🎨 Personal Portfolio Website

> A beginner-friendly portfolio template built with Bootstrap 5

---

## 📌 What Is This?

A **Personal Portfolio Website** — your digital business card that showcases who you are, your skills, projects, and how to contact you. All in one place!

**Built With:** HTML5 • CSS3 • JavaScript • Bootstrap 5.2.3

---

## 🎯 Features

| Feature | Description |
|---------|-------------|
| 🏠 **Homepage** | Hero section with your name and introduction |
| 📄 **Resume** | Display education, experience, and skills |
| 💼 **Projects** | Gallery to showcase your work |
| 📬 **Contact** | Form for visitors to reach you |
| 📱 **Responsive** | Works on desktop, tablet, and mobile |
| 🌙 **Dark Theme** | Modern dark design with teal & purple accents |

## 🏗️ How It Works

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│    HTML     │ ──▶ │    CSS      │ ──▶ │ JavaScript  │
│ (Structure) │     │  (Styling)  │     │ (Behavior)  │
└─────────────┘     └─────────────┘     └─────────────┘
       │                   │                   │
       └───────────────────┴───────────────────┘
                           │
                           ▼
                  ┌─────────────────┐
                  │  Your Website   │
                  └─────────────────┘
```

**Simple Explanation:**
- **HTML** = The skeleton (what's on the page)
- **CSS** = The clothes (how it looks)
- **JavaScript** = The muscles (what it does)

## 🎨 Customization Cheat Sheet

| Change This | Edit This File | Look For |
|-------------|----------------|----------|
| Your name | `index.html` | `<h1>` in hero section |
| Your title | `index.html` | `<h3>` under name |
| About text | `index.html` | About section `<p>` tags |
| Colors | `css/styles.css` | `:root` variables |
| Profile photo | `index.html` | `<img src="">` tag |

### Color Variables (in styles.css)

```css
:root {
    --bs-primary: #14b8a6;    /* Teal accent */
    --bs-secondary: #8b5cf6;  /* Purple accent */
    --bs-body-bg: #0a0a0a;    /* Background */
}
```

## 🔧 Troubleshooting

| Problem | Solution |
|---------|----------|
| Styles not loading | Check `<link>` path to `css/styles.css` |
| Images broken | Use relative paths: `assets/image.jpg` |
| Changes not showing | Hard refresh: `Ctrl+Shift+R` |
